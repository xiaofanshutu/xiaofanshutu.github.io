练习题

假如还有 59 天放假，问：合 xx 个星期零 xx 天

public class App {
    public static void main(String[] args) {
        int days = 258;
        int weeks = days/7;
        int leftDays = days%7;
        System.out.println(days + "天," + "合" + weeks + "星期零" + leftDays + "天");
    }
}
258天,合36星期零6天
